TASK: Recommend validated measurement instruments for each variable and/or build a custom survey instrument.

FOR EACH VARIABLE REQUIRING MEASUREMENT:

Option A — Validated Instrument Recommendation:
- Instrument name and abbreviation
- Number of items and response format
- Constructs measured (subscales if applicable)
- Scoring instructions (sum vs. mean, reverse scoring, subscale structure)
- Reliability (Cronbach's α — report published values)
- Validity evidence (what validation studies support this instrument)
- APA 7th edition reference
- Sample item (1 example)
- Why this instrument fits the current operationalization

Option B — Custom Instrument Builder:
If no validated instrument fits or the researcher needs a custom scale:
- Draft 5-10 Likert items per construct (5-point or 7-point scale)
- Include reverse-scored items (at least 20% of scale)
- Specify response anchors explicitly
- Note that custom scales need pilot testing for reliability

INSTRUMENT BANK PRIORITY (prefer validated over custom):
- Anxiety: GAD-7, STAI, BAI
- Depression: PHQ-9, BDI-II, CES-D
- Stress: PSS-10, DASS-21
- Self-esteem: Rosenberg Self-Esteem Scale, RSES
- Personality: BFI-44, BFI-2, TIPI
- Loneliness: UCLA Loneliness Scale (3rd ed.)
- PTSD: PCL-5, IES-R
- Social desirability: Marlowe-Crowne (short form)
- Life satisfaction: SWLS (Diener et al.)
- Mindfulness: FFMQ, MAAS

SURVEY STRUCTURE GUIDELINES:
- Always include demographics section at the end (not beginning — reduces dropout)
- Include attention check items (1-2 per 50 items)
- Add estimated completion time at the top
- Include informed consent text at the beginning
- Add debriefing text at the end

Output the complete survey instrument, ready to deploy.
